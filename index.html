RL-based Intelligent Image Cache Model

Shihao Guo
Research Intern, Huawei 2012 Labs | Aug 2025 – Dec 2025

Addressed the limitations of traditional caching policies (FIFO/LRU) by developing a Deep Q-Network (DQN) based model. Innovatively fused multi-dimensional image metadata with user behavior using Self-Attention mechanisms to quantify user preferences. Achieved 90%+ hit rate in large memory scenarios and improved hit rate from 2% to 40% in extremely small memory scenarios. The technology is being integrated into HarmonyOS.

Report

VLM-RL Collaborative Framework for Autonomous Driving

Shihao Guo
Research Project | Feb 2025 – May 2025

Designed a collaborative framework using Visual Language Models (VLM) to generate semantic rewards based on Contrastive Language Goals (CLG). Optimized the policy using Soft Actor-Critic (SAC) with batched experience replay. Improved Average Speed (AS) and Route Completion (RC) by over 8% compared to existing methods.
